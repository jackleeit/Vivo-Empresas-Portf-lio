Serviços de Gestão

Gestão é uma ferramenta para gerenciar o uso e o consumo das linhas e dos pacotes contratados por uma empresa. Este gerenciamento pode ser realizado em cada linha individualmente ou em várias linhas organizadas em grupos.
Ao contratar este serviço, o gestor recebe um login e uma senha de acesso a um site onde todas as configurações de gestão são realizadas. Atualmente, o serviço Gestão realiza o controle dos tráfegos de voz.

CARACTERÍSTICAS


  • Configuração de listas autorizadas e listas não autorizadas.
  • É possível definir números curtos para discagem abreviada.
  • É possível gerenciar o consumo dos pacotes contratados e do valor monetário para o excedente.
  • É possível realizar os seguintes bloqueios: chamadas externas, chamadas para determinados grupos, para operadoras de Longa Distância, para chamadas de Longa Distância Nacionais e Internacionais, para chamadas em roaming internacional e para chamadas recebidas a cobrar.
  • É possível controlar as chamadas por horário e calendário.

BENEFÍCIOS


  • Controle de gastos: é possível configurar o limite de excedente para cada linha da conta.
  • Otimização: é possível distribuir os minutos compartilhados entre as linhas da conta para aperfeiçoar a utilização.
  • Segurança: só o gestor da conta tem acesso às configurações das linhas.
